## 4. 使用

- 安装依赖 `requirements.txt`
- `dlib` 安装不了 使用 `pip` 安装 `docs/dlib~.whl`
- 运行 `Clever_Sign.py`
- 不用可能大概率，就是有 bug \(￣︶￣*\))
